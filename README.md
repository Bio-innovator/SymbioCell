# SymbioCell

A high-performance cellular automata simulation exploring endosymbiotic mutations, multicellular emergence, and self-organizing biological network dynamics.

一个基于元胞自动机的大尺度高性能多细胞内共生演化模拟器。

---

## 🧬 生物组件系统说明 (Cellular Component System)

在 `SymbioCell` 的世界中，独立的单细胞通过繁衍、变异与抱团，能够涌现出具备复合功能的“多细胞有机体”。各元胞类型各司其职：

* 🟢 **Green - 光能细胞**：活在光照层（上半场）。提供光合作用能量，是整个生态系统的底层能量生产者。
* 🟣 **Purple - 化学能细胞**：活在厌氧层（下半场）。提供厌氧代谢能量，在阴影区疯狂繁衍。
* 🔵 **Blue - 运动细胞**：具备物理位移能力。能够自发成链，产生机械推力，负责推动整艘“多细胞战舰”前行。
* 🔴 **Red - 捕食细胞**：掠食者。贪婪地吞噬周围所有的杂色生命细胞。
* 🟠 **Orange - 支架细胞**：多细胞组织的“外骨骼”与结构支撑。为内部脆弱的能量细胞提供结构保护与空间防线。
* 🟡 **Yellow - 溶酶清理细胞**：生态系统的清洁工。专门游走在死亡高发区，高效率分解净化细胞尸体，重新腾出宝贵的生存空间。
* ⚫ **Corpse - 细胞尸体**：生命凋亡后的有机残留。会触发特定的向内腐烂或被黄色细胞分解的代谢判定。

## ⏳ 版本说明 (Version Introduction)

### 📌 v1.1 - Shockwave (双色爆炸爆发版)
* **版本特点**：绿色的蓝藻细胞和紫色的古菌细胞拥有绝对统治力。由于没有代谢限制，它们开局就会像超新星爆发或者狂暴的藻华一样，呈几何级数向外进行侵略式的疯狂扩散（Shockwave）。
* **核心改动**：基础的一维数组单循环 CA 架构，解除蓝色运动元胞的全局游走限制，实现初步的物理推动效果。
* **演化缺陷**：属于典型的“单向消耗模型”。非繁衍细胞只减不增，演化后期必然走向全场饱和或灭绝的“生态热寂”；且由于双缓冲颠倒，画面渲染存在滞后性，边界处存在元胞自杀黑洞。

### 📌 v1.2 - Root (内共生突变与结构失衡版)
* **版本特点**：打破了孤立单细胞的壁垒，首次引入了 **33.3% 的器官分化突变** 与 **33.3% 的连带邻居爆发式组织复制**。细胞开始学会抱团繁衍，形成了复杂的“多细胞组织群落”。
* **核心改动**：
    * **架构神级优化**：底层彻底重构为 2D 嵌套循环，剔除百万次高能耗的取模和除法计算，实现运行时零内存分配，彻底斩断 GC（垃圾回收）卡顿。
    * **全新视口渲染**：上线了动态裁剪画布技术，支持大空间拖拽滚动条，以及 `Ctrl + 鼠标滚轮` 视野微调缩放。
* **演化缺陷**：**遭遇“塑料垃圾式大污染”**。由于橙色支架细胞和黄色清理细胞在这个版本里属于“长生不老”的状态，缺乏空间与能量消耗成本。这导致演化到中后期，整个皿会被大面积蔓延开来的橙色和黄色细胞死死堵满，生态系统彻底固化锁死（Root）。

### 📌 v1.3 - Red (巨噬猎群与1像素永生迷宫版)
* **版本特点**：画面全线重构为高端护眼暗黑微光生物风格。修改了红色的内讧门槛（周围同类数量 $\ge 4$ 才触发内讧自残），让红色的“捕食巨噬细胞”在全场大放异彩。
* **核心改动**：
    * **实装代谢成本**：橙色支架外壳加入了“必须依赖绿/紫宿主供能”与“过于密集凋亡”的限制；黄色清洁工加入了“尸体密度不足时因饥饿自毁”的判定。
    * **物理性能解放**：限制蓝色最大推力为 12 格，修复了“蓝色无法紧密尾随跟车”的物理锁死。支持无缝双轴同步等比鼠标缩放。
* **演化缺陷**：**意外卡出“红魔死局”数学奇观**。红色捕食细胞在疯狂繁衍中展现出惊人的自组织几何选择性，完美利用规则漏洞，在全场拉出了一条条宽度精准为 1 像素的**“永生丝状拓扑迷宫网络”**。在这种网络中，没有任何一个红色细胞的邻居数会达到 4 个，它们从而变得不可战胜，锁死了全场生态，并在大后期因超长无尽扫描引发了线程过载卡顿。
